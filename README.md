# 🖼️ AI Image Recognition System

An AI-powered Image Recognition System built using Flask and TensorFlow (MobileNetV2).

## 🚀 Features

- Upload Image
- Drag & Drop Upload
- AI Image Recognition
- Top 5 Predictions
- Confidence Score
- Image Details
- Prediction Time
- Prediction History
- PDF Report Download
- Responsive UI

## 🛠 Technologies Used

- Python
- Flask
- TensorFlow
- MobileNetV2
- HTML
- CSS
- JavaScript
- ReportLab

## ▶️ How to Run

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
